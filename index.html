Privacy Policy

This application is an online download manager and media player.

Information Collection and Use

This app uses Firebase Authentication to allow users to sign in. Basic account information such as email address or user ID is collected only to authenticate users and to store personal download history.

User-Provided Content

Users can provide custom download links. These links are used only to perform the requested download and to display download history.

Third-Party Services

This app uses Firebase Authentication provided by Google. No data is shared for advertising or analytics purposes.

Data Security

Data is transmitted securely using HTTPS.
